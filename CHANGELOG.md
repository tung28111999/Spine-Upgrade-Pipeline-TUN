# Changelog

## 0.1.0 - Initial Public Release

- Added PySide6 desktop UI for upgrading exported Spine runtime assets.
- Added headless command-line runner.
- Added automatic source Spine version detection.
- Added folder, `.zip`, and `.rar` input support.
- Added batch processing for multiple Spine assets.
- Added Unity-ready runtime export packaging.
- Added atlas repacking with near-square page ratio settings.
- Added public release docs, example configs, build scripts, and MIT license.
